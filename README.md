# 📅 スケジュール管理 Discord Bot

Google カレンダーと連携して、Discord サーバーにスケジュールを自動表示・通知する Bot です。

---

## ✨ 機能

### カレンダー表示
- 指定チャンネルに **カレンダー埋め込み** と **ステータス埋め込み** の2件を固定投稿
- 先月・来月ボタンで月切り替え（ephemeral表示）
- 今日の日付をハイライト表示（📍）
- 曜日別カラードット（🔴日・🔵土・▫️平日）

### 予定管理（ロール制限あり）
| 操作 | 方法 |
|------|------|
| ➕ 追加 | ボタン → モーダル入力 |
| ✏️ 編集 | ボタン → 予定選択 → モーダル入力 |
| 🗑️ 削除 | ボタン → 予定選択 → 確認 |
| 🔄 更新 | ボタン（全員操作可） |

- 時刻は `2100` / `21:00` / `2500`（翌1時）など柔軟な入力に対応

### 通知設定
- 予定ごとに **複数の通知** を設定可能
- 通知対象: **ロール** / **ユーザー** / `@everyone` / `@here`
- 通知タイミング: 5分前〜24時間前などから選択
- 通知送信先は `NOTIFY_CHANNEL_ID` で別チャンネルに分離可能

### 操作ログ（Audit Log）
- 予定の追加・編集・削除を **専用チャンネル** に色分けEmbedで記録
  - 🟢 追加 / 🟡 編集 / 🔴 削除

### UX
- 操作結果はすべて **ephemeral（自分だけ見えるメッセージ）** で通知
- 成功メッセージは5秒、エラーメッセージは8秒でカウントダウン後に自動削除
- 編集フロー完了後、選択メニューを自動削除

## 🛠️ セットアップ

### 必要なもの
- Node.js v18 以上
- Discord Bot（アプリケーションページで作成）
- Google Cloud サービスアカウント（Google Calendar API 有効化済み）

### インストール

```bash
cd src
npm install
```

### 環境変数

ルートまたは `src/` に `.env` ファイルを作成してください。

```env
# Discord
DISCORD_TOKEN=           # Bot トークン
DISCORD_CHANNEL_ID=      # カレンダーを投稿するチャンネルID

# Google Calendar
GOOGLE_CLIENT_EMAIL=     # サービスアカウントのメールアドレス
GOOGLE_PRIVATE_KEY=      # サービスアカウントの秘密鍵（\n を含む文字列）
GOOGLE_CALENDAR_ID=      # 対象カレンダーのID（通常はメールアドレス形式）

# 権限・チャンネル設定
OPERATOR_ROLE_NAME=CalendarOperator   # 操作を許可するロール名
NOTIFY_CHANNEL_ID=                    # 通知送信先チャンネルID（未設定なら DISCORD_CHANNEL_ID を使用）
LOG_CHANNEL_ID=                       # 操作ログ送信先チャンネルID（未設定でログなし）

# Cron（省略可）
CRON_SCHEDULE=*/5 * * * *             # 同期間隔（デフォルト: 5分ごと）
```

### 起動

```bash
cd src
node index.js
```

## 📁 ファイル構成

```
src/
├── index.js       メインエントリ・インタラクションハンドラ・cron管理
├── calendar.js    Google Calendar API ラッパー
├── embed.js       Discord Embed・コンポーネントのビルダー
├── modals.js      モーダル（追加・編集フォーム）のビルダー
├── storage.js     状態・通知設定の永続化（JSONファイル）
├── notifier.js    cron定期実行による通知チェック・送信
└── logger.js      コンソールログのファイル記録（日別ローテーション）

data/
├── state.json     Bot状態（メッセージID・最終ハッシュ等）
└── notices.json   通知設定（eventId ごとの通知リスト）

logs/
└── YYYY-MM-DD.log  日別ログファイル（7日分保持、自動削除）
```

## 🔐 権限設計

| ロール | 操作可能な機能 |
|--------|--------------|
| **管理者**（Administrator権限） | すべての操作 |
| `OPERATOR_ROLE_NAME` のロール保持者 | 追加・編集・削除・通知設定 |
| **全員** | 🔄 更新ボタン・◀▶ 月切り替えボタン |

---

## 📌 備考

- Google カレンダーの予定変更は最大5分後に自動反映されます（cron間隔による）
- 通知は予定開始から24時間経過後に自動クリーンアップされます
- メインチャンネルへの投稿は常に2件のみ（カレンダー・ステータス）に制限されています
- Discord インタラクショントークンは15分間有効です
- コンソールログは `logs/YYYY-MM-DD.log` にも保存されます（7日分を自動保持）
